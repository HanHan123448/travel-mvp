## 1. 设计理念

### 1.1 核心关键词

| 关键词 | 设计映射 |
|--------|---------|
| 小众 | 非主流审美、克制留白、拒绝信息过载 |
| 发现感 | 瀑布流不确定性、每张卡片不同高度、惊喜下滑 |
| Z世代 | 暗黑模式优先、emoji表情符号、大胆渐变、圆角亲和 |
| Ins风 | 图片优先、大图小文、卡片阴影、毛玻璃效果 |
| 隐秘 | 深色背景模拟"探索未知"的沉浸感 |

### 1.2 设计原则

- **图片为王**: 封面图占卡片面积 ≥ 65%，文字为辅助
- **暗黑不压抑**: 深色背景 + 高亮点缀色，保持视觉轻盈
- **触达优先**: 所有交互区域 ≥ 44px（移动端友好）
- **少即是多**: 每屏信息不超过3条卡片，留白即呼吸

## 6. 卡片组件设计

### 6.1 卡片结构

```
┌─────────────────────┐
│                     │
│                     │
│    📷 封面图片       │  ← 图片区：占卡片高度 60-70%
│    (圆角 12px)      │     图片上叠加渐变遮罩 (底部→深色)
│                     │     收藏按钮 ❤️ 悬浮在右上角
│              ┌────┐ │
│              │ ❤️  │ │  ← 收藏按钮：圆形44px，半透明背景
│              └────┘ │     未收藏: 🤍 灰色，已收藏: ❤️ 琥珀色
├─────────────────────┤
│                     │
│  胶片摄影  📚 独立书店│  ← 标签行 (最多2个，超出用"+N")
│                     │
│  藏在胡同里的        │  ← 标题 (最多2行，超出省略号)
│  胶片书店            │
│                     │
│  📍 北京            │  ← 城市 + 时间 (底部元信息)
│  2周前              │
└─────────────────────┘
```

### 6.2 卡片尺寸

| 属性 | 值 | 说明 |
|------|-----|------|
| 宽度 | 自适应列宽 | `(屏幕宽 - 16*2 - 12)/2` |
| 图片高度 | 由原始图片比例决定 | 瀑布流天然高度不一 |
| 图片最小高度 | 140px | 防止纯文字卡片 |
| 图片最大高度 | 280px | 防止单张图过高 |
| 信息区高度 | ~72px | 标签 + 标题 + 城市时间 |
| 卡片圆角 | 12px | |
| 卡片背景 | `#1A1A2E` | |
| 卡片内边距(信息区) | 12px | |

### 6.3 图片区遮罩

```
┌─────────────────────┐
│ 图片原始区域          │  ← 图片完全填充
│                     │
│ ░░░░░░░░░░░░░░░░░░ │  ← 上半部：无遮罩/微透明
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │  ← 下半部：渐变遮罩
│ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │     rgba(15,15,26,0) → rgba(15,15,26,0.7)
└─────────────────────┘
```

遮罩目的：保证收藏按钮和底部过渡的可见性。

### 6.4 卡片交互状态

| 状态 | 视觉表现 | 触发条件 |
|------|---------|---------|
| Default | 卡片背景 `#1A1A2E`，无边框 | 页面静止 |
| Hover/Press | 背景微亮 `#1E1E35`，边框 `1px solid rgba(168,85,247,0.3)` | 手指/鼠标悬停 |
| Active (点击瞬间) | Scale 0.97，持续150ms | 手指按下未释放 |
| 进入详情 | 卡片作为共享元素过渡到详情页 | 点击 |
| 收藏按钮点击 | ❤️ 缩放弹跳动画 (300ms) | 点击收藏按钮 |

### 6.5 卡片加载态（骨架屏）

```
┌─────────────────────┐
│ ████████████████████ │  ← 灰色脉冲矩形 (图片占位)
│ ████████████████████ │
│ ████████████████████ │
├─────────────────────┤
│ ████████  ████████  │  ← 标签占位
│ ██████████████████  │  ← 标题占位
│ ██████████████      │
│ ██████  ██████      │  ← 元信息占位
└─────────────────────┘
  骨架元素颜色: #2A2A40
  脉冲动画: opacity 0.3 → 0.6 → 0.3 (1.5s infinite)
```


| 城市数据来源 | 从 `travel_item.city` DISTINCT 动态获取 |

### 7.4 标签网格

| 属性 | 值 |
|------|-----|
| 布局 | 3列网格，列间距8px，行间距8px |
| 每个标签Chip | 文字居中，emoji + 标签名，内边距 10px 0 |
| 未选中 | 背景 `#2A2A40`，文字 `#B0B0C8` |

| 多选支持 | 标签可多选，选中不限数量 |
| 数据来源 | 从 `tag` 表全量加载 |
